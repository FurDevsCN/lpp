# 欢迎啦！在这里你将学到怎么编译这个项目。

## 直入主题

```
$ make build # 正常
$ make debug # debug专用，没事别用
$ ./lpp --help # 查看帮助
```

## 参数说明

**-i** 或 **--interactive** 可打开一个可交互控制台。尽情实验你的代码吧！  
**-h** 或 **--help** 可获取帮助。  
**-v** 或 **--version** 可显示版本。
**filename** 可以直接执行脚本  
**-** 可以让你从**标准输入**读取脚本并执行\(比如：`lpp - < test.lpp`)。
