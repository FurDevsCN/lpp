## 特点

### 超简单

可以很方便地使用L++地各种接口，并且不需要知道它的内部原理！

### 只使用STL标准库

不使用任何第三方库\(**最小版本**\)，只要目标平台支持C++ STL就可以直接跑！

### 高扩展性

只要向函数表添加一个**lambda函数**就可以完成创建自定义命令。  
请读 [如何开发（未完成）](docs/develop.md) 来获取相关帮助。
